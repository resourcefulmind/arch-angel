# arch-angel

Persistent architectural memory for engineering teams.

## What is this?

AI coding tools retrieve context and generate code — but they don't understand your architecture, track your design decisions, or notice when your codebase drifts from its intended design.

arch-angel fills that gap: persistent architectural memory, evidence-backed answers, and drift detection.

## Status

Early development. V1 in progress.

## Setup

```bash
pnpm install
pnpm dev scan <path>
```
